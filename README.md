# Curso de Phyton regressivo XD

## 📒 Descrição
Projeto para o bootcamp da DIO - Python AI Backend Developer - Em que utilizo IA generativa.
Esta é uma demonstração bastante simples, mas que aplica um conceito da IA: A capacidade de levar as pessoas além quando se trata de potencial de pesquisa, conhecimento e produção.

## 🤖 Tecnologias Utilizadas
- ChatGPT
- GIT

## 🧐 Processo de Criação
Após concluir o bootcamp "Python AI Backend Developer", eu solicitei ao ChatGPT, via prompt, que criasse um curso de Python. Meu prommpt foi o seguinte:

"Crie um curso, como se você fosse um cientista da computação, que ensine a usar a linguagem python (do básico ao avançado), ensine a fazer gestão de versionamento de códigos usando o GIT, ensine a realizar manipulação e consulta à banco de dados usando o SQL , ensine a desenvolver APIs para integrações sistêmicas e ensine à consumir via API serviços de inteligência artificial disponíveis gratuitamente"

O objetivo foi verificar o quão próximo (ou melhor) a sugestão de cusrso criada pela IA poderia ficar, comparado ao curso que acabei de estudar.
